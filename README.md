
## Reverse Vending Machine

Developed a smart reverse vending machine prototype using **Raspberry Pi 4** and Arduino for automated recyclable item detection and collection.

The project was built using recycled and scrap materials to demonstrate a sustainable approach toward waste management. The system integrates image processing, sensors, and motor control for automated operation.

### Hardware Used:
- Raspberry Pi 4
- Arduino Microcontroller
- Servo Motor
- DC Motor (Recycled from CD Drive)
- Camera Module for Image Processing
- Ultrasonic Sensor

### Components Description:

- **Raspberry Pi 4**  
  Used as the main processing unit for running image processing algorithms, system control, and communication with hardware components.

- **Arduino**  
  Used for real-time control of motors, sensors, and hardware interfacing.

- **Camera Module**  
  Used for capturing images and performing image-based object detection and classification.

- **Ultrasonic Sensor**  
  Used for distance measurement and detecting the presence of objects during the vending process.

- **Servo Motor**  
  Used for controlling the mechanical movement mechanism of the system.

- **DC Motor (CD Drive Motor)**  
  Repurposed from scrap materials and used for mechanical movement inside the vending mechanism.

### Key Features:
- Automated recyclable item detection
- Image processing-based identification
- Sensor-based object detection
- Sustainable design using recycled materials
- Embedded hardware and software integration
